== Description ==

This plugin shows user's last login time in backend. Can be useful for website admin.
Plugin is tested upto wordpress version 4.9.1 and compatible with lower versions as well.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress


== Frequently Asked Questions ==

= What is the use of plugin? =

This plugin has been created to know when the user has logged in last time on website.

= What are the future aspects of this plugin? =

Plugin enhancement is in progress to advance its feature so that is can be used on most of the wordpress sites.
